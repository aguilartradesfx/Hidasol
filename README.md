# Hidasol - Sistema de Gestión de Órdenes

A sophisticated production order management system for a print shop, featuring hierarchical organization, real-time status tracking, and printable production tickets.

## Features

- **Dashboard**: Real-time statistics with percentage changes
- **Hierarchical Browser**: Navigate orders by Year → Month → Week → Day
- **Advanced Search**: Filter by ID, client, product, status
- **Order Management**: Create, edit, delete with validation
- **Status Tracking**: Quick-toggle status updates
- **Production Tickets**: Print-optimized boletas

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui + Radix UI
- Local Storage

## Development

```bash
npm install
npm run dev
```
